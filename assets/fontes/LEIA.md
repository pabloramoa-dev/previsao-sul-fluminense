# Fontes

Coloque aqui dois arquivos:

- `bold.ttf` - fonte bold para numeros e titulos
- `regular.ttf` - fonte regular para textos

Se estes arquivos nao existirem, `gerar_carrossel.py` e `gerar_story.py`
usam o fallback `DejaVuSans` (disponivel no runner do GitHub Actions),
entao a automacao funciona mesmo sem fontes customizadas.

Sugestoes de fontes gratuitas para uso comercial: Montserrat, Inter, Roboto.
